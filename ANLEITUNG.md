# 🚀 Anleitung: Website öffnen

## 📋 Schnellstart

### Option 1: Entwicklungsserver (Empfohlen)
```bash
# Im project Ordner
npm run dev
```
Die Website öffnet sich automatisch unter: `http://localhost:3000`

### Option 2: Produktions-Build
```bash
# Im project Ordner
npm run build
npm run preview
```
Die Website öffnet sich unter: `http://localhost:4173`

### Option 3: Direkt öffnen
1. Öffnen Sie die Datei `project/index.html` in Ihrem Browser
2. Die Website lädt automatisch mit allen Assets

## 🌐 Online verfügbar

Die Website ist auch online verfügbar unter:
- **Entwicklung**: http://localhost:3000
- **Produktion**: http://localhost:4173 (nach Build)

## 📱 Responsive Testen

Die Website ist vollständig responsive. Testen Sie sie auf:
- **Desktop**: 1200px+
- **Tablet**: 768px - 1199px  
- **Mobile**: 320px - 767px

## 🔧 Troubleshooting

### Falls die Website nicht lädt:
1. Stellen Sie sicher, dass Sie im `project` Ordner sind
2. Führen Sie `npm install` aus
3. Starten Sie den Server mit `npm run dev`

### Falls Assets nicht geladen werden:
1. Führen Sie `npm run build` aus
2. Starten Sie dann `npm run preview`
